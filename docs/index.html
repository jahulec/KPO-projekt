Generator stron dla artystów

Darmowe narzędzie do tworzenia statycznych stron internetowych (HTML/CSS/JS) dla twórców.

Uruchom generator Instrukcja PDF Releases (ZIP) Wideotutoriale

Jak zacząć

  1. Otwórz index.html w przeglądarce.
  2. Wybierz profil i sekcje, uzupełnij treści.
  3. Wybierz styl (Flagowe lub Basic).
  4. Wyeksportuj stronę do ZIP.

Publikacja strony

Najprościej: GitHub Pages. Wgraj wyeksportowane pliki do repozytorium i włącz Pages (Deploy from branch: main, folder /).

Generator to osobny projekt. Wyeksportowana strona to gotowa paczka do hostingu (GitHub Pages / Netlify / dowolny serwer).

Finansowanie (KPO / NextGenerationEU)

Przedsięwzięcie jest realizowane dzięki wsparciu z Krajowego Planu Odbudowy i Zwiększania Odporności oraz finansowane przez Unię Europejską w ramach NextGenerationEU.

Szczegóły w pliku kpo.html.
